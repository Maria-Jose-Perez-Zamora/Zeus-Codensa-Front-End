B. Flujo por rol
La palabra flujo viene del latín fluxus, que significa fluir o moverse.
Un flujo es el camino que sigue un usuario para completar una acción.
Ejemplo:
Flujo de inscripción de jugador
Inicio
 → Registro
 → Crear perfil deportivo
 → Indicar posición
 → Marcar disponibilidad

Flujo de capitán
Login
 → Crear equipo
 → Invitar jugadores
 → Completar equipo
 → Subir comprobante de pago

Flujo del organizador
Login
 → Crear torneo
 → Definir fechas
 → Programar partidos
 → Registrar resultados







1. Gestión de Usuarios
Este módulo permite que los participantes se registren dentro del sistema y creen su perfil deportivo.
Entre las funcionalidades principales se encuentran:
registro de jugadores


creación del perfil deportivo


selección de posición de juego


disponibilidad para integrarse a un equipo


Este módulo facilita que los capitanes puedan encontrar jugadores disponibles para completar sus equipos.
2. Gestión de Equipos
Este módulo permite que los capitanes creen y administren sus equipos.
Las funcionalidades incluyen:
creación de equipos


asignación de nombre y escudo


invitación de jugadores


visualización de la alineación del equipo


Los mockups muestran cómo los capitanes pueden gestionar la composición de su equipo dentro de la plataforma.





3. Gestión del Torneo
Este módulo está dirigido principalmente a los organizadores del torneo.
Permite:
crear torneos


configurar fechas importantes


definir reglas y reglamentos


programar partidos


Esto centraliza la organización del torneo dentro de la plataforma.
4. Gestión de Partidos
Los mockups también muestran las pantallas donde se podrán visualizar los partidos programados y registrar resultados.
Las funcionalidades incluyen:
consulta del calendario de partidos


visualización de equipos participantes


registro de marcadores


registro de tarjetas y goleadores

5. Estadísticas y Resulados
Este módulo permite consultar información relevante del torneo, como:
tabla de posiciones


historial de partidos


máximos goleadores


Estas estadísticas se calcularán automáticamente a partir de los resultados registrados en el sistema.
